# SpiceClock 0.1.1 (2026-05-29)

## 概要
- SpiceClock v0.1.1 は v0.1.2 に向けた refactor-package-engine-api ブランチの途中リリースです。
- 透明ウィンドウの可変表示領域、時計/ビジュアライザのレイヤー順、設定画面の操作性を改善しました。

## 主な変更
- ビジュアライザ透明ウィンドウを表示内容に追従する可変サイズに調整。
- 時計/日付レイヤーをビジュアライザレイヤーより上に固定。
- 設定ウィンドウの前回タブと表示位置を記憶。
- カラーパレットの表示位置を記憶し、表示中は設定ウィンドウを操作できないモーダル動作に変更。
- Windows に登録されているフォント一覧から時計フォントを切り替えるように変更。
- 時刻、日付、ビジュアライザの非表示設定を追加。
- 設定ウィンドウタイトルに Cargo.toml のバージョンを表示。

## 注意
- v0.1.2 は main マージ時に付与予定です。このリリースは v0.1.1 として公開します。
