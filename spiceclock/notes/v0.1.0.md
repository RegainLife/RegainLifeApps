# SpiceClock 0.1.0 (2026-05-21)

## 概要
- SpiceClock の初回公開リリースです。
- Clock / Date / Visualizer / Image / Scene3D を独立Widgetとして扱う、Windows 常駐型のデスクトップ HUD エンジンです。

## 主な機能
- Direct3D 12 / DirectComposition によるクリック透過の透明オーバーレイ表示。
- WASAPI loopback で取得した PC 内部音に反応するプロシージャル・リングビジュアライザ。
- Clock / Visualizer は内部結合せず、Workspace 単位で必要な処理のみを起動。
  - 時計単体 Workspace では Audio / Visualizer の重い処理は走りません。
- `workspaces / clock_themes / visualizer_asset / audio_profile / performance_profile` を分離保存。
- Quality Governor が GPU フレーム予算に応じて bloom / blur / particle / 内部スケール / FPS を段階的に調整。
- Music State Machine（Verse / Buildup / Drop / Silent）によるブレンド遷移。

## インストール
- `SpiceClock.Setup.exe` を実行してインストールしてください。
- インストール直後はウィンドウが他のアプリの背面に回ることがあります。タスクトレイの SpiceClock アイコン経由で前面表示を切り替えてください。

## 既知の課題
- 無音アイドル時の GPU 消費が想定より高い場面があります。
- Style 詳細画面の「サウンド感度」項目は名称・実装ともに調整中です。
