SmartScreen警告が出る場合

SpiceClock.Setup.exe は現在コード署名を行っていないため、Microsoft Defender SmartScreen によって 「Windows によって PC が保護されました」と表示される場合があります。

1. 「詳細情報」をクリック
最初の画面では「実行」ボタンが表示されないため、「詳細情報」をクリックします。
2. ファイル名を確認して「実行」
表示内容を確認し、問題なければ「実行」をクリックします。

不安な場合は、ダウンロードしたファイルの SHA-256 がリリース表の値と一致しているか確認してください。 公式配布元はこのページと GitHub Releases です。
