CalendarJP

日本の祝日表示可能で詳細な天気も表示できるカレンダーです。
Windows 11 のタスクバー時計クリックで開き、純正の Windows カレンダーフライアウトを置き換えます。

CalendarJPSetup.exe でインストールし、Windows の「インストールされているアプリ」からアンインストールできます。自動アップデートも同じ setup exe を使って適用します。

タスクトレイに常駐し、トレイアイコンの右クリック →「スタートアップに登録」で Windows 起動時に自動起動できます。
「終了」するとフックが解除され、純正の Windows カレンダーに戻ります。

Google カレンダー（iCal URL）との連携、気象庁データによる天気表示（降水ナウキャスト含む）にも対応しています。
※ .NET 9 ランタイムが必要です。
