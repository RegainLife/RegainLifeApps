CalendarJP

Windows向け日本の祝日カレンダーアプリ

CalendarJPは、日本の祝日に対応した無料のカレンダーアプリです。 Windows 11 の純正カレンダーを違和感なく置き換えて、祝日・天気・Google カレンダーの予定をひとつの画面で確認できます。

通常の操作と同様にタスクバー時計クリックで開き、アプリ本体はタスクトレイに常駐し、トレイアイコンの右クリック →「スタートアップに登録」で Windows 起動時に自動起動できます。 「終了」すると純正の Windows カレンダーに戻ります。

Google カレンダー（iCal URL）との連携、気象庁データによる天気表示（降水ナウキャスト含む）にも対応しています。 v1.4以降からインストール式になり、自動アップデートにも対応しました。

※ .NET 9 ランタイムが必要です、見つからない場合は自動でインストールするか確認があります。

SmartScreen警告が出る場合

CalendarJPSetup.exe は現在コード署名を行っていないため、Microsoft Defender SmartScreen によって 「Windows によって PC が保護されました」と表示される場合があります。

1. 「詳細情報」をクリック
最初の画面では「実行」ボタンが表示されないため、「詳細情報」をクリックします。
2. ファイル名を確認して「実行」
表示内容を確認し、問題なければ「実行」をクリックします。

不安な場合は、ダウンロードしたファイルの SHA-256 がリリース表の値と一致しているか確認してください。 公式配布元はこのページと GitHub Releases です。
