# v1.3.0 — 2026-05-01

## 新機能

### 天気表示

ヘッダーに現在地の天気情報を表示するようになりました。

- **天気アイコン・気温** — 気象庁の予報 API と AMEDAS 観測値をもとに表示
- **降水コメント** — JMA ナウキャストタイルを解析し「現在 小雨」「25分後 雨」などをリアルタイム判定
- **自動地域取得** — Windows の位置情報サービスで現在地を自動判定
- **手動設定** — 設定画面で地域名を検索・登録可能

### アプリアイコン刷新

アプリアイコンおよびタスクトレイアイコンを CalendarJP 専用デザインに変更しました。

## 改善

- 設定画面のボタンホバー色を統一（半透明白、青塗りつぶしを廃止）
- 設定画面のデザインセクション項目順を整理（背景透過度 → 背景色 → セル形状 → グリッド線 → ...）
