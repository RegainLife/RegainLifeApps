# v1.4.0 — 2026-05-01

## 追加

- 自己完結型の `CalendarJPSetup.exe` によるインストール方式を追加
- Windows の「インストールされているアプリ」からアンインストールできるように対応
- setup exe を使った自動アップデート方式に対応

## 改善

- 新規インストールと自動アップデートの配布ファイルを `CalendarJPSetup.exe` に統一
- アンインストール時にインストール先・ショートカット・レジストリ・一時ファイルを削除
- 設定ファイルを `Documents\CalendarJP_Setting\settings.json` に保存する方式へ変更
- 設定画面で「現在の天気」と「気温観測」の参照先を分けて表示
- Windows 位置情報の取得を高精度・短いキャッシュに変更

## 修正

- 手動地域を検索した場合は自動取得をオフにして、指定地域を優先するように変更
