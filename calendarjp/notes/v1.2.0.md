# CalendarJP v1.2.0 リリースノート

**リリース日**: 2026年4月30日

## 変更点

### デザインカスタマイズ機能を追加

設定画面（⚙）から外観を自由にカスタマイズできるようになりました。

- **背景の透過度** — スライダーで 0〜100% を調整。0% にするとウィンドウ枠・文字・仕切り線のみ残り、完全透過になります
- **セルの形状** — 四角・丸を選択できます
- **グリッド線** — カレンダーグリッドの区切り線を表示・非表示にできます。色もパレットで変更可能
- **カラーパレット** — 背景色・アクセント色（今日・選択日）・祝日色・土曜日色をそれぞれ 12 色のパレットから選択できます

### 予定インジケーターの改善

予定がある日の表示をわかりやすく改善しました。

- 日付の下部に **カラーバー（下線）** で表示するようになりました（以前は小さな点）
- 複数の予定がある日は予定件数に応じて **色分けされた複数バー** で表示します（1件目＝アクセント色、2件目＝黄、3件目＝緑）
- 祝日は祝日色のバーで表示します
- 選択中・今日の日付ではインジケーターが白く反転して見やすくなります

### UI の細部改善

- 設定画面のスクロールバーをダークテーマに統一しました
- 今日の日付ボタン（ヘッダー部分）のホバー時に文字が見えなくなる問題を修正しました
